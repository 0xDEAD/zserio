## Constants

A constant is an immutable named value. The syntax and behavior is similar to C or C++. Their syntax is as
follows:

```
const base-type NAME = literal;
```

**Example**

```
const bit:1 FALSE = 0;
const bit:1 TRUE = 1;
const int16 i = 1234;
const int32 j = -5678;
```
